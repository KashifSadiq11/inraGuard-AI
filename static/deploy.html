Deployment Steps

Step 1

Install HuggingFace Hub Library

Install the official HuggingFace Python library that lets you interact with Spaces via CLI.

pip install huggingface_hub --break-system-packages
If you are using a virtual environment, remove --break-system-packages
Step 2

Verify Your HuggingFace Username

Use your token to confirm your HuggingFace username before proceeding.

python3 -c "
from huggingface_hub import HfApi
api = HfApi(token='YOUR_HF_TOKEN')
user = api.whoami()
print(user['name'])
"
Expected output: your HuggingFace username (e.g. Kashif77)
Step 3

Create the HuggingFace Space

Create a new Docker-based Space in your HuggingFace account. Replace YOUR_USERNAME and YOUR_HF_TOKEN.

python3 -c "
from huggingface_hub import HfApi
api = HfApi(token='YOUR_HF_TOKEN')
api.create_repo(
    repo_id='YOUR_USERNAME/infraguard-ai',
    repo_type='space',
    space_sdk='docker',
    private=False,
    exist_ok=True
)
print('Space created successfully')
"
Expected output: Space created successfully
Step 4

Upload All Application Files

Upload every required file from your local project directory to the Space. Replace YOUR_HF_TOKEN, YOUR_USERNAME, and /path/to/infraguard with your actual values.

python3 -c "
from huggingface_hub import HfApi
import os

api = HfApi(token='YOUR_HF_TOKEN')
base = '/path/to/infraguard'

files = [
    'main.py',
    'requirements.txt',
    'Dockerfile',
    'agents/__init__.py',
    'agents/base.py',
    'agents/specialists.py',
    'agents/tools.py',
    'static/index.html',
    'static/docs.html',
]

for f in files:
    full_path = os.path.join(base, f)
    if os.path.exists(full_path):
        api.upload_file(
            path_or_fileobj=full_path,
            path_in_repo=f,
            repo_id='YOUR_USERNAME/infraguard-ai',
            repo_type='space',
            token='YOUR_HF_TOKEN'
        )
        print(f'Uploaded: {f}')
    else:
        print(f'SKIPPED (not found): {f}')
"
Expected output: Uploaded: main.py, Uploaded: requirements.txt ... (all 9 files)
If a file shows "No files modified" — it was already uploaded and unchanged. That is fine.
Step 5

Add Environment Secrets

Add your Groq API key and model name as secrets. These are stored securely and injected as environment variables at runtime.

python3 -c "
from huggingface_hub import HfApi
api = HfApi(token='YOUR_HF_TOKEN')

api.add_space_secret(
    repo_id='YOUR_USERNAME/infraguard-ai',
    key='GROQ_API_KEY',
    value='YOUR_GROQ_API_KEY'
)
api.add_space_secret(
    repo_id='YOUR_USERNAME/infraguard-ai',
    key='GROQ_MODEL',
    value='llama-3.3-70b-versatile'
)
print('Secrets added successfully')
"
Expected output: Secrets added successfully
Never hardcode API keys in your code or Dockerfile. Always use secrets.
Step 6

Check Build Status

Monitor the Space build status. It takes 2–3 minutes to build the Docker image.

python3 -c "
from huggingface_hub import HfApi
api = HfApi(token='YOUR_HF_TOKEN')
space_info = api.get_space_runtime(repo_id='YOUR_USERNAME/infraguard-ai')
print('Stage:', space_info.stage)
"
Stage ValueMeaning
BUILDINGDocker image is being built — wait 2-3 minutes
RUNNINGApp is live and ready
STOPPEDApp stopped — check logs on HuggingFace UI
ERRORBuild failed — check Dockerfile and requirements.txt

Your App URL

Once Stage shows RUNNING, your app is live at:

https://YOUR_USERNAME-infraguard-ai.hf.space

Replace YOUR_USERNAME with your actual HuggingFace username (e.g. kashif77-infraguard-ai.hf.space).

Important: Dockerfile Port

HuggingFace Spaces requires port 7860. Make sure your Dockerfile uses this port:

EXPOSE 7860
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "7860"]
Using port 8000 will cause the app to not respond. Always use 7860 for HuggingFace Spaces.

How to Update After Code Changes

When you make changes to your code, re-upload the changed files using the same upload command from Step 4. HuggingFace will automatically rebuild.

# Re-upload a single changed file
python3 -c "
from huggingface_hub import HfApi
api = HfApi(token='YOUR_HF_TOKEN')
api.upload_file(
    path_or_fileobj='/path/to/infraguard/main.py',
    path_in_repo='main.py',
    repo_id='YOUR_USERNAME/infraguard-ai',
    repo_type='space',
    token='YOUR_HF_TOKEN'
)
print('Updated successfully')
"
